This file describes the Planetz simulator from an "end-user" perspective. For details regarding the code and implementation, please refer to technical.html

Running the project
Build and run the project. Initially you will only see the sun. You can interact with the tool using the following commands:

  • A - adds a new planet
  • D - deletes a planet
  • SPACE - toggles animation
  • TAB - switches between planets. selected planet is shown in dark blue.
  • UP/DOWN - moves selected planet further/closer to the Sun.
  • T - toggles trails.

What does the project do?
The project demonstrates a simple hierarchy. The planets revolve around the sun and the moons around the planet. The speed and distance between objects is randomly chosen.
