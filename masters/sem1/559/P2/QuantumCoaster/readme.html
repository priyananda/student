This file describes the Roller Coaster simulator from an "end-user" perspective. For details regarding the code and implementation, please refer to technical.html

Running the project
Build and run the project. Make sure you set the current working directory to 'bin' if you are debugging.
The simulator has two basic modes: RUN and EDIT. In EDIT mode you can change the track. In RUN mode you sit back and watch the coaster fly! In RUN mode, you have two camera modes, GOD and TRAIN. In GOD mode you can move forward/backward, turn horizontally and vertically. In TRAIN mode you have no freedom and see the world as if you are on the train.

Common Keys

  • Press ESC to quit program
  • Press UP/DOWN to move forward/backward in GOD camera mode
  • Press LEFT/RIGHT to turn left or right in GOD camera mode
  • Press PGUP/PGDOWN to look up/down in GOD camera mode

Run Mode Keys

  • Press 'R' to go into EDIT mode
  • Press 'F' to toggle GOD/TRAIN camera
  • Press SPACE to toggle train animation
  • Press 'P' to start/reset train
  • Press '+'/'-' to increase/decrease speed
  • Press '<'/'>' to increase/decrease bogeys in the train
  • Press 'S' to toggle shadows

Edit Mode Keys

  • Press 'R' to go into RUN mode
  • Press 'A'/'D'/'W'/'S'/'Q'/'Z' to move selected control point in X,Z,Y directions
  • Press TAB to switch to next control point
  • Press 'N' to add new control point
  • Press DELETE to delete a control point

Editing the Track
You can click on a control point to select it(it will change its color to white). You can then click-and-drag using the LEFT mouse button to move it in XZ(horizontal) direction. You can click-and-drag using the RIGHT mouse button to move it vertically(Y).
