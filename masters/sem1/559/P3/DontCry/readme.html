This file describes the graphics town simulator from an "end-user" perspective. For details regarding the code and implementation, please refer to technical.html

Running the project
Build and run the project. Make sure you set the current working directory to 'bin' if you are debugging.
You can move in the world in first person camera mode, with a combination of mouse and keyboard. You can enter and exit the portal at locations specified using the ENTER key. SPACE triggers special effects like the Matrix effect in specified locations.

How to move in the world

  • Press UP or W,DOWN or S to move forward/backward
  • Press LEFT or A,RIGHT or D to strafe left/right
  • Move mouse horizontally to turn
  • Move mouse vertically to look up/down
